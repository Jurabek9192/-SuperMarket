# RFM-анализ клиентов и дашборд продаж для Superstore

## Краткое описание
Проект демонстрирует анализ продаж и клиентов сети Superstore.  
Используется методика RFM (Recency, Frequency, Monetary) для сегментации клиентов и выявления приоритетных групп для маркетинговых действий.  
Создан интерактивный дашборд в Power BI с ключевыми метриками, графиками и рекомендациями.

## Цели проекта
- Показать ключевые показатели продаж: Выручка, Прибыль, Маржа, Количество заказов, Средний чек  
- Сегментировать клиентов на основе RFM  
- Выявить топ-продукты и топ-города по выручке  
- Проанализировать динамику продаж и прибыльность  
- Сформулировать рекомендации для маркетинговых кампаний  

## Описание данных
- Источник: Superstore dataset  
- Основные таблицы и колонки:
  - **Sales**: CustomerID, OrderID, OrderDate, SalesAmount, Profit, Category, City..  
  - **RFM Customers**: CustomerID, Recency, Frequency, Monetary, R_Score, F_Score, M_Score, RFM Score, RFM Segment  

## Методы анализа
- Очистка и подготовка данных  
- Расчёт RFM показателей:
  - Recency — дни с последней покупки  
  - Frequency — количество заказов  
  - Monetary — сумма покупок  
- Присвоение R, F, M баллов (1–5) и формирование сегментов  
- Создание дашборда с визуализацией: карты, Combo Chart, Heatmap, Bar Charts, KPI Cards  

## Основные KPI и визуалы
- Выручка, Прибыль, Маржа, Количество заказов, Средний чек  
- Топ-10 товаров и городов  
- Новые и повторные клиенты  
- ARPU (средний доход на клиента)  
- RFM Heatmap и сегментация  
- Combo chart: Выручка vs Прибыль  

## Рекомендации и бизнес-выводы
- **Чемпионы** → VIP-акции  
- **Лояльные** → Upsell / Cross-sell  
- **Потенциально лояльные** → стимулировать повторные покупки  
- **Под угрозой** → email/SMS реактивация  
- **Потерянные** → Win-back / исключение из кампаний  

## Инструкция по просмотру
1. Открыть файл `.pbix` в Power BI Desktop  
2. На главной странице дашборда находятся ключевые KPI и фильтры  
3. Использовать слайсеры для выбора периода, категории или сегмента клиентов  

## Стек технологий
- Power BI Desktop  
- DAX (Calculated Columns, Measures)  
- Excel / CSV (для исходных данных)
